# Contributing

Thanks for your interest in contributing!

## How to Contribute

- Open an issue to discuss bugs or feature ideas.
- Submit a pull request with clear scope and tests when possible.

## Development Setup

- Install dependencies:

```sh
npm install
```

## Tests

- Lua tests:

```sh
busted
```

- Lint:

```sh
luacheck .
```

## Documentation

- Docs are built with VitePress.

```sh
npm run build
```

## Pull Request Checklist

- Keep changes focused and small.
- Update docs or types where appropriate.
- Add or update tests when behavior changes.

## Commit Messages

- Use Conventional Commits 1.0.0 (e.g., `feat: ...`, `fix: ...`, `chore: ...`).
- Spec: https://www.conventionalcommits.org/en/v1.0.0/
